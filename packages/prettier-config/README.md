## `@lens-protocol/prettier-config`

Project's [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ pnpm add -D @lens-protocol/prettier-config
```

**Install peer dependencies**

```bash
$ pnpm add -D prettier
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@lens-protocol/prettier-config"
}
```
